Descubra as áreas

BEM-VINDO

Esse site irá te explicar como funciona as fórmulas para descobrir a área de uma figura geométrica plana e calcular o resultado para você!

triângulo retângulo

Para encontrar a área de um triângulo retângulo, basta dividir por 2 o resultado da multiplicação da base (b) pela altura (h)

Círculo

A área de um círculo é pi(π) vezes o raio(R) elevado ao quadrado (A = π r²)

Quadrado

Para encontrar a área de um quadrado, basta multiplicar os dois lados (L)

losango

Para encontrar a área de um losango, basta multiplicar a base (b) pela altura (h)

TRAPÉZIO

Para encontrar a área de um trapézio, basta somar as bases (B + b) vezes a altura (H) dividido por dois

triângulo equilátero

a área de um triângulo equilátero é dada por

A = (L² . √3) ÷ 4

triângulo (heron)

Para encontrar a área de um triângulo usando apenas a medida dos lados, utiliza-se a fórmula de heron

A = √p . (p - a).(p - b).(p - c)
